Salty Boy

This site serves as a wrapper for the SaltyBoy project to access historical matches for the fighters on Salty Bet which is an automated M.U.G.E.N game. For more information about the project see the GitHub, also there you can find the API endpoints.

If you want you can use the Chrome Extension created that interfaces with this website (this is still a WIP, link will be dropped here #soon).

Disclaimer

While the webserver is meant to be pretty light and currently the information can be fetched by anyone it is intended a small side-project. So this means to please not abuse the API. Additionally, it is in a constant state of flux and will change randomly without warning. Should you want to create your own interface/extension by using the information in this database please be aware of this fact as I may randomly break things for you, also you must be compliant with the MIT License found on the GitHub repo. If you have suggestions or improvements you would like to suggest please start an issue on the GitHub repository.
